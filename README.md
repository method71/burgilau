# Perfectum Prime калькулятор

Статический HTML-калькулятор стоимости установки приточного клапана для публикации через GitHub Pages.

## Структура

```text
.
├── index.html
├── README.md
└── .gitignore
```

`index.html` автономный: стили, скрипты и изображения встроены в один файл. Папка `assets` не требуется.

## Локальный запуск

Откройте `index.html` в браузере двойным кликом.

Если нужен локальный сервер:

```bash
python -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

## Публикация через GitHub Pages

1. Загрузите файлы из этого проекта в репозиторий `method71/burgilau`.
2. Откройте настройки репозитория на GitHub.
3. Перейдите в `Settings -> Pages`.
4. В блоке `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`.
6. Сохраните настройки.

После публикации сайт будет доступен по адресу:

```text
https://method71.github.io/burgilau/
```
